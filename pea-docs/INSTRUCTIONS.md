## 🏛️ 3 הרשויות — כל אחת נפרדת

### 1️⃣ PEA — הראשי (Provincial Electricity Authority)
**מקומי:** PEA Koh Phangan (Thong Sala) — `การไฟฟ้าส่วนภูมิภาค อำเภอเกาะพะงัน`
**טלפון:** 077-377-018
**מה מגישים:** חבילת `pea-85kwp.html` + `structural-letter.html`
**פורמט:** פיזי בנייר, בסניף המקומי
**עלות:** אין אגרה להודעה. אגרה ~2,000-5,000 THB להחלפת מונה דו-כיווני בסיום.

### 2️⃣ ERC — מקביל וחובה (Energy Regulatory Commission)
**פורטל:** https://app04.erc.or.th/eformpublic/
**מה מגישים:** `erc-notification.html` + מצורפים
**פורמט:** אונליין (PDF upload)
**עלות:** חינם
**זמן:** הודעה פשוטה, לא אישור. מקבלים אסמכתא תוך 7 ימים.

### 3️⃣ רשות מקומית (เทศบาลตำบลเกาะพะงัน)
**אם הגג יכול להיחשב שינוי מבני** — לשלוח עותק של `structural-letter.html`
**ברוב המקרים:** פטור מהיתר בנייה תחת `กฎกระทรวง 2564` — רק להודיע ולשמור העתק

**⚠️ חשוב:** בלי חתימה + חותמת + מספר רישיון של מהנדס רשום — **PEA דוחה את החבילה באופן אוטומטי**. זו נקודת הכישלון מס' 1 של זרים.

## 📋 Checklist הגשה (לפי סדר)

### שלב 1 — הכנת מסמכים (לפני הגשה)
- [ ] מלא את כל ה-Fields הפתוחים ב-`pea-85kwp.html` (שם לקוח, ת.ז., כתובת, מספרי PEA וכו')
- [ ] המר HTML → PDF (Chrome → Print → Save as PDF)
- [ ] הדפס עותק פיזי ×2 (אחד ל-PEA, אחד שומרים)
- [ ] חתום מהנדס חשמל מורשה על Doc 2 (SLD) ו-Doc 3 (Equipment Spec)
- [ ] חתום מהנדס מבנים על `structural-letter.html`
- [ ] הכן מצורפים:
  - [ ] העתק ת.ז./פספורט של הבעלים
  - [ ] העתק `หนังสือรับรอง` של החברה (≤3 חודשים)
  - [ ] העתק `โฉนด` (Land Title Deed) או חוזה שכירות
  - [ ] חשבון חשמל PEA אחרון
  - [ ] תמונות הגג הקיים (4 זוויות + שם הרחוב ברקע)

### שלב 2 — הגשה ל-PEA Koh Phangan
- [ ] גש פיזית ל-PEA Thong Sala
- [ ] בקש: *"ขอแบบคำขอเชื่อมต่อระบบ Solar Rooftop Self-Consumption 85 kWp แบบไม่ขายคืน"*
- [ ] מלא את הטופס **הרשמי של PEA** (המכתב שלנו = תומך בלבד)
- [ ] צרף את כל החבילה + מצורפים
- [ ] קבל מספר תיק (`เลขที่รับ`) — שמור

### שלב 3 — הודעת ERC (במקביל)
- [ ] הירשם ב- https://app04.erc.or.th/eformpublic/
- [ ] בחר: `แจ้งการประกอบกิจการพלังงานที่ได้รับการยกเว้น`
- [ ] העלה PDF של `erc-notification.html` + מצורפים
- [ ] קבל `เลขที่รับแจ้ง` — שמור

### שלב 4 — ביקור שטח של PEA
- [ ] מהנדס PEA מגיע לאתר (~14–30 ימים לאחר הגשה)
- [ ] בודק התקנה, מדידות, Zero-Export device
- [ ] חותם על `รายงานการตรวจสอบ`

### שלב 5 — החלפת מונה + COD
- [ ] PEA מחליפים את המונה לדו-כיווני (כדי לראות שאין export)
- [ ] אגרה: ~2,000–5,000 THB
- [ ] קבל `หนังสืออนุญาตเชื่อมต่อ` (אישור חיבור רשמי)
- [ ] הפעלה מסחרית (COD)

### שלב 6 — עדכון ERC לאחר COD
- [ ] חזור לפורטל ERC
- [ ] העלה תמונות מערכת + אסמכתא PEA
- [ ] קבל `ใบรับแจ้ง` רשמי

## 🚫 טעויות נפוצות שצריך להימנע מהן

1. **הגשה ללא חתימת מהנדס** → דחייה מיידית
2. **שימוש באינוורטר שלא ב-PEA approved list** → דחייה. Huawei SUN2000-75KTL-M0 ✅ ברשימה
3. **הצהרת Zero-Export בלי מכשיר פיזי** → מבקרי PEA בודקים פיזית. DTSU666-H חובה
4. **דילוג על ERC notification** → PEA עלולים לנתק את המערכת אחרי הפעלה
5. **Voc מחרוזת > 1,100V** → דחייה טכנית. התכנון שלנו 643V STC ✓ בטוח
6. **DC/AC ratio > 1.4** → דחייה טכנית. שלנו 1.13 ✓
7. **POA חסר חותמת חברה של Grantor** → החזרה לתיקון
